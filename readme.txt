Make the dashboard Solarized

== Description ==
Adds an admin color scheme which not only colorizes the admin menu and
the admin bar but the whole dashboard.
Use it if you know the dashboard, if you know the location of every part
of dashboard beacause Solarized's contrast is lower thanother color schemes
to ease your eye.


== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip the plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
